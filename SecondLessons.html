LESSON 1&2: INTRODUCTION TO CSS Lessons 1 and 2 are all about the introduction of CSS, like the different layers it has, (such as content, behaviour and presentation) and overrall use. this is also where the three linking styles are introduced, inline, embedded, and external. LESSON 3: STYLE SHEETS This lesson gives an overview of how CSS essentially works and how it's formatted. Examples of different style sheets can be also seen here. The lesson also explains why CSS code is formatted this way, or its syntax. The formatting on this table is this code's output.
LESSON 4: SELECTORS, DIV, SPAN, AND LINK. This lesson introduces us to the following; selectors, which include classes, DIV, a block level element that adds structure to your webpage, span, which does the same thing but for inline elements, and link, which is used in external stylesheets.

example of using the SPAN tag.

LESSON 5: SELECTORS, DIV, SPAN, AND LINK. This lesson is a continuation from the last lesson, expanding it further by expanding selectors into selecting multiple of them, grouping, and nesting them. In this lesson, we also learn to set dimensions and how to use pseudo classes.

LESSON 6: FORMATTING TEXT WITH CSS Lesson 6 showcases different ways for you to format text, from its weight and size, to its dimensions and variants and so much more. It basically shows you how to use CSS in decorating the text of your website.

This text uses text formatting.
LESSON 7: FORMATTING LAYOUT WITH CSS This lesson introduces the Box Model, which is a way for you to understand formatting layout in CSS. It includes margins, borders, padding and the actual content. CSS allows you to control the dimension of the box using the height and width properties.
